# archives-kmu
## 국민대학교 기록물관리위원회 웹페이지 개설
* 국민대학교 기록물관리위원회 시스템부에서 기획 및 제작한 웹페이지입니다. 

### 개발 과정
1. Figma를 통해 웹 UI 디자인 진행
2. Html, Css, Javascript를 통해 개발

### 콘텐츠
* 국민대학교의 기록물들을 더욱 쉽고 재밌게 접할 수 있도록 반응형 콘텐츠로 제작
* 제작한 콘텐츠들은 한달에 한번씩 퀴즈 형태로 간식 이벤트 진행
